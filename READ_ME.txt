This is a simple Twitter Crawler developed in Java for academic research purposes.

1. Setting Search Queries
- The twiiterQuery.config file has to be in the same folder as the executable jar file
- The languages property can be empty
- The keywords property must be set. At least one keyword has to be set. 

2. Authentication keys
- The oauthKeys.config file has to be in the same folder as the executable jar file
- You have to set the keys of your application in the oauthKeys.config
- You have to create your own application in dev.twitter.com to get your Twitter App Oauth keys.
- No Oauth keys - no application running.
